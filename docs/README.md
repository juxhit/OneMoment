# Документация OneMoment

## О проекте

OneMoment — веб-приложение для сбора и показа фотографий на мероприятиях (свадьбы, корпоративы, конференции). Гости сканируют QR-код, загружают фото с телефона; организатор управляет событием из админ-панели и выводит Live Wall на экран.
